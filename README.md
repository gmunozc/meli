# Frontend Test

MercadoLibre Frontend Test

## Getting Started

### Requisites

Install:
- docker
- docker-compose

### Running

Run:
```
docker-compose up
```
or running in the background
```
docker-compose up -d
```

Open http://localhost:3000 to view it in the browser.
